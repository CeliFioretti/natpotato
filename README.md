# NATPotato

NATPotato es una aplicación web simple para la gestión de tareas y notas personales.  
Permite a los usuarios crear tareas que pueden marcarse como completadas o pendientes, así como crear notas para guardar información de todo tipo de forma organizada y privada.

## Funcionalidades

- Registro y autenticación de usuarios
- Creación, edición y eliminación de tareas
- Marcado de tareas como completadas o pendientes
- Creación, edición y eliminación de notas
- Acceso a la información de forma privada por usuario

## Tecnologías utilizadas

- Next.js
- TypeScript
- Tailwind CSS
- API Routes (App Router)
- Base de datos relacional SQL
- Autenticación con tokens

## Diseño del sistema

El proyecto cuenta con:
- Diagrama entidad-relación (ER)
- Casos de uso
- Endpoints REST documentados
- Definición de payloads JSON
